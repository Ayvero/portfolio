Proyectos Java

En esta sección se encuentran proyectos que abarcan una amplia gama de técnicas avanzadas en programación, a traves de los cuales se dejan ver habilidades sólidas en la resolución de problemas complejos, optimización de recursos y análisis de estructuras de datos.

Algunas de las habilidades adquiridas:

  • Backtracking
  • Algoritmos Greedy
  • Manejo de Grafos
  • Programación Dinámica
  • Optimización de Recursos
  • Algoritmos de Caminos Mínimos
  • Solución de Problemas Combinatoriales
  • Estructuras de Datos
  • Análisis y Resolución de Problemas

Proyecto 1: Implementación de Clases GrafoDirigido y GrafoNoDirigido

Implementacion de clases que representan grafos dirigidos y no dirigidos, permitiendo manipular estructuras de datos para resolver problemas complejos en teoría de grafos.

Proyecto 2: Recorridos en Grafos - Depth-First Search (DFS) y Breadth-First Search (BFS)

Desarrollo de implementaciones de algoritmos de búsqueda en profundidad y en anchura, fundamentales para explorar grafos y encontrar caminos.

Proyecto 3: Cambio de Monedas - Algoritmo de Programación greedy

Resolución del problema del cambio de monedas utilizando programación dinámica, optimizando la cantidad mínima de monedas necesarias para un valor dado.

Proyecto 4: Asignación de Tareas a Procesadores - Optimización de Recursos

Implementación de un algoritmo para la asignación eficiente de tareas a procesadores, minimizando el tiempo total de ejecución.

Proyecto 5: Maximizar el Número de Actividades Compatibles

Desarrollo de una solución para maximizar el número de actividades compatibles que se pueden realizar en un mismo día.

Proyecto 6: Algoritmo de Dijkstra para el Camino Más Corto en un Grafo

Implementacion del algoritmo de Dijkstra para encontrar el camino más corto entre nodos en un grafo ponderado.

Proyecto 7: Problema del Viajante - Solución Aproximada en Grafos Ponderados

Resolución del problema del viajante utilizando un algoritmo aproximado de greedy para encontrar una ruta eficiente en un grafo ponderado.

Proyecto 8: Laberinto - Camino de Longitud Mínima en una Matriz. Algoritmo backtracking.

Desarrollo de una solución para encontrar el camino de longitud mínima en un laberinto representado por una matriz.

Proyecto 9: Conjunto de Salas - Recorrido Máximo en un Grafo Dirigido. Backtracking

Implementacion de un algoritmo para encontrar el recorrido máximo en un conjunto de salas conectadas representadas como un grafo dirigido.

Proyecto 10: Suma de Subconjuntos - Combinaciones que Suman un Valor Dado. Backtracking

Resolución del problema de la suma de subconjuntos encontrando todas las combinaciones que suman un valor dado.

Proyecto 11: Partición de Conjunto en Subconjuntos de Suma Igual. Backtracking

Implementación de un algoritmo para particionar un conjunto en dos subconjuntos con sumas iguales.

Proyecto 12: Caballo de Atila - Recorrido Cerrado en una Matriz

Resolución del problema del recorrido cerrado de un caballo de Atila en una matriz utilizando un algoritmo de backtracking.

Proyecto 13: Tablero Mágico - Suma Igual en Filas y Columnas

Implementación de un tablero mágico en el que las sumas de las filas y columnas son iguales, utilizando técnicas avanzadas de programación.

Proyecto 14: Pirámide de Números - Suma de Casillas

Desarrollo de una pirámide de números donde la suma de las casillas sigue una estructura definida, utilizando algoritmos de programación dinámica.

Proyecto 15: Ordenar Tablero 4x4 - Resolución de Puzzle

Implementacion de un algoritmo para resolver un puzzle de ordenación en un tablero 4x4, optimizando el número de movimientos necesarios.

Proyecto 16: Asignación de Tareas a Procesadores - Minimización del Tiempo Total

Desarrollé un algoritmo de asignación de tareas para procesadores, con el objetivo de minimizar el tiempo total de ejecución.
